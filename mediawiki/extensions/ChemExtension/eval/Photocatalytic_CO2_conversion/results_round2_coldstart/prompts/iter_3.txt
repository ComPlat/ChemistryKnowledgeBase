FIELD-BY-FIELD EXTRACTION GUIDELINES  
(These rules directly address the most common errors.)

1. Concentrations (cat conc, PS conc, e-D conc, additives conc, H-D conc)  
   • Report in millimolar (mM).  
   • Convert units when needed:  
     – µM → divide by 1000 (e.g. 25 µM → 0.025 mM)  
     – nM → divide by 1 000 000  
   • Give the numeric value only, no unit.

2. Solvents and solvent-ratio  
   • List each solvent exactly as printed (e.g. “CH3CN”, “DMF”).  
   • Put the composition in solvent-ratio using “A:B” or “A:B:C” form (numbers only).

3. Feedstock gas  
   • Record only the gas species (and mixture if stated).  
   • Omit percentages, flow rates, and pressures.  
     Example: “100 % CO2 (1 atm)” → “CO2” ; “CO2/Ar 10:90” → “CO2/Ar (10:90)”.

4. Temperature  
   • Numeric °C value only (no unit symbol).

5. λexc  
   • Keep wavelength in nm plus any light-source descriptor exactly as written, e.g.  
     “450 nm (LED, 100 mW cm-2)”.

6. irr time  
   • Express in hours (h).  
   • Convert minutes: 10 min → 0.167 ; seconds: 30 s → 0.0083.

7. Light intensity  
   • Copy the numeric value *and* its unit as given (e.g. “100 mW cm-2”, “1 W”).

8. Turnover number (TON) fields  
   • Record the numeric value only, no unit.

9. Turnover frequency (TOF) fields  
   • Report in h⁻¹.  
   • If the paper gives s⁻¹, multiply by 3600 (e.g. 0.47 s⁻¹ → 1692 h⁻¹).

10. Quantum yield fields  
    • Give the numeric percentage without the “%” sign (e.g. “2.3” not “2.3 %”).

11. H-D column  
    • Copy the label the authors use for isotopic experiments (e.g. “D2O”, “H2O”).  
      If no isotopic substitution is involved, leave blank.

12. Order of precedence when multiple values are reported for the *same* experiment:  
    • Prefer the main text over SI; if both list the same experiment, use the main text values.
