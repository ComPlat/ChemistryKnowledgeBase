GENERAL RULES  
1. Keep every field; never add or delete keys.  
2. Create one object per distinct photocatalytic experiment. “Distinct” means at least one of cat, PS, e-D, solvent ratio, λexc, additives, temperature, or irradiation time differs.  
3. If a value is genuinely absent, write an empty string "" (do NOT write “not reported”, “n/a”, 0, or null).  
4. Copy wording exactly as printed—spellings, capitalisation, punctuation—except where numeric normalisation is required (see below).  
5. Do not guess or average; only transcribe explicitly provided data.  
6. Never merge data from different tables/figures unless the paper states they are identical conditions.

FIELD-SPECIFIC GUIDANCE  
• cat – Full catalyst name or formula exactly as written.  
• cat conc – Report in mM. If given in µM divide by 1000 (25 µM→0.025). Numeric string only, no unit.  
• PS – Full photosensitiser name exactly as written.  
• PS conc – Report in mM (convert µM÷1000). Numeric string only.  
• e-D – Sacrificial electron donor name.  
• e-D conc – Report in M; convert from mM or % (v/v) only when the paper gives enough information. Numeric string only.  
• solvent-ratio – Reproduce exactly, e.g. “DMF/TEOA (4:1)”.  
• additives – Any additional reagents or atmospheres not part of the solvent (e.g. “0.1 M BIH”, “CO2/Ar (10:90)”).  
• feedstock gas – Gas used for the catalytic run (e.g. “CO2”, “CO2/Ar (10:90)”).  
• Temperature – Numeric value in °C without unit.  
• λexc – Keep full wording including “nm” and light source or power if present (e.g. “450 nm (LED, 100 mW cm-2)”, “420–650 nm”).  
• intensity – Use only when the paper lists optical power density separately; copy with units (e.g. “100 mW cm-2”). Leave blank if this information already sits inside λexc.  
• irr time – Illumination time in hours; numeric only.  
• Turnover_number__CO / __H2 / __HCOOH – Plain number; keep scientific notation if that is how the paper writes it (e.g. “1.7×10^4”).  
• Turnover_frequency__CO / __H2 – Numeric value as reported; remove unit text.  

ROW COVERAGE STRATEGY  
1. Start with the main photocatalytic performance table; include every entry.  
2. Add screening/optimisation series, time-course points, and control experiments (no cat, no PS, dark, Ar, etc.) provided they list at least one numeric product metric.  
3. When a parameter is stated once for a whole series (e.g. “cat = 0.025 mM for all entries”) apply it to every row in that series.  
4. Create separate rows when different irradiation times map to distinct TON/TOF values.  

NUMERIC & UNIT NORMALISATION  
• Keep all significant digits; never round.  
• Convert µM→mM (÷1000).  
• Convert mM→M (÷1000) when filling e-D conc.  
• Temperature stays numeric without “°C”.  
• Do not add units to numeric-only fields.
